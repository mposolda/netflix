You have successfully installed the Netflix portlet based on GateIn portal, JBoss Richfaces, JBoss Portlet Bridge, Netflix Java client

You can search for movies and movie people (directors, actors) and show informations about them. Informations are taken from Netflix with usage of Netflix api.

Please note that for proper work of this portlet, you need to provide your Netflix api credentials (consumerKey and consumerSecret), which needs to be provided as system properties to JVM where GateIn portal is running. Needed steps are:

  • Obtaining your api credentials after registration on http://developer.netflix.com/member/register.
  • Adding this line:

    JAVA_OPTS="$JAVA_OPTS -Dnetflix.consumer.key=YOUR_CONSUMER_KEY -Dnetflix.consumer.secret=YOUR_CONSUMER_SECRET"

    to the end file $GATEIN_HOME/bin/standalone.conf . Replace YOUR_CONSUMER_KEY and YOUR_CONSUMER_SECRET with proper values.
  • Restarting GateIn portal
